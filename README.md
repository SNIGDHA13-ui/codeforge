
# CodeForge 🚀

CodeForge is a full-stack competitive programming platform inspired by LeetCode.  
It allows users to solve coding problems, submit solutions, and track their progress.

---

## 🛠 Tech Stack

**Frontend**
- React
- Tailwind CSS
- Monaco Editor

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

---

## ✨ Features (Planned)

- 🔐 User Authentication (JWT based)
- 📚 Problem Listing with Difficulty Levels
- 🧠 Code Editor with Syntax Highlighting
- ⚡ Code Execution Engine
- 📊 Submission History
- 🏆 Leaderboard & Streak System (Future)

---

## 🚧 Project Status

Currently under development.  
Backend setup in progress.
